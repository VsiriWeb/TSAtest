We used Bootstrap as the framework for our website to streamline the design and development process. With Bootstrap’s responsive grid system, we were able to create a layout that adapts seamlessly to different screen sizes, ensuring a smooth experience for users on both mobile and desktop devices.

Citation: "MIT OpenCourseWare."
